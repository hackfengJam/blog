## 前言

程序员的自我修养：

1. 一切语言、技术或者框架，本质都是工具，工具的价值在于为使用者提供竞争优势。

2. 如果真的有一种语言或者框架很牛逼，那么荣耀或者赞誉应该属于创造它的人，与使用者没有半毛钱关系；
使用者的荣耀应该来自；针对恰当的需求使用恰当的语言或者框架，并做到按时交付以及高质量。

3. 大部分人并不是天生有选择恐惧症，也不是天生的杠精，在我看来所有关于选择的迷惑或者争吵，大都因为：
    1. 没有设定清晰的标准；  
    2. 标准不唯一；  
    3. 标准之间没有优先级或者权重；
